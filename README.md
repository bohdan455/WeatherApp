# WeatherApp
## Prerequisites
### Before we begin, please make sure that you have the following installed on your system:

* Visual Studio 2022 or later
* .NET SDK 7 or later
* MySQL Server and MySQL Workbench (or any other MySQL client)
## Step 1: Clone the repository
Clone or download the repository from GitHub to your local machine.

## Step 2: Configure the database connection
Navigate to the TodoApi/appsettings.json file and provide your database connection string and openweathermap api.

## Step 3: Open the project in Visual Studio
Open the WeatherApp project in Visual Studio by clicking on the TodoApi.sln file.

## Step 4: Run the application
Press F5 to launch the application in debug mode. The WeatherApp should start running and listening.

And that's it! You have successfully built and run the TodoApi application.
